# gdoc-to-medium

Drag a Google Doc into a Drive `Ready to Publish` folder; a scheduled local job converts it to
Medium markdown, creates a Medium draft for review, then files the doc under `Published/`.

This is a stub. Setup, usage, and limitations will be documented as the project is built out.
See `docs/superpowers/specs/` for the design and `docs/superpowers/plans/` for the build plan.
